# interface/

TypeScript-интерфейсы, используемые в 2+ файлах проекта.

## Содержимое

Пока папка пуста. Интерфейсы добавляются по мере роста проекта.

## Конвенции

- Один файл = один домен (например, `order.ts`, `product.ts`).
- Имя файла — в нижнем регистре, отражает сущность.
- Только `interface`, не `type` (типы живут в `types/`).

## Использование

```typescript
import type { OrderItem } from "~/interface/order";
```
